This is JWT Secured Authentication gateway made for Quantum as assignment.
Made using MERN/

Steps to Install:
1. The root directory contains two directories:
	Client
	Server
2. Client is having ReactJS Files.
3. Server is having NodeJS FIles.
4. Install node modules:
	Go to terminal > cd client > npm i
	Go to terminal > cd server > npm i
5. Run the Development server:
	Go to terminal > cd client > npm start
	Go to terminal > cd server > npm start
6, The backend is powered in 3001 port, where as ReactJS is powered on port 3000.
